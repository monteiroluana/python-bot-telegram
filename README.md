# Python Telegram Bot
**Python Telegram Bot**: Application developed to study the [Telegram_API](https://core.telegram.org/api) using the Python language.


## Getting start

### create a .env file whith
```
BOT_TOKEN=
OPENWEATHERMAP_TOKEN= 
```

### environment
```shell
$ pyenv local 3.8.10

$ pipenv shell

$ pipenv sync --dev
```



### run
```shell
$ python manage.py migrate
$ python bot.py
```
